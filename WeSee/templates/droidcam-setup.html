DroidCam Setup Guide

Use your Android phone as a high-quality webcam for object detection

Step 1: Install DroidCam App

On Your Android Phone:
  • Open Google Play Store
  • Search for "DroidCam Wireless Webcam"
  • Install the app by Dev47Apps
  • Grant camera and microphone permissions
On Your Computer:
  • Download DroidCam Client from dev47apps.com
  • Install the DroidCam client software
  • Restart your computer after installation

Step 2: USB Connection Setup

Phone Settings:
  • Connect phone to computer via USB cable
  • Enable "Developer Options" in Settings
  • Turn on "USB Debugging"
  • Set USB connection to "File Transfer (MTP)" mode
  • Allow USB debugging when prompted
Computer Setup:
  • Open DroidCam Client on computer
  • Select "USB" connection method
  • Click "Start" to establish connection
  • You should see your phone's camera feed

Step 3: Enable Developer Options (Android)

How to enable Developer Options:
  1. Open Settings on your Android phone
  2. Scroll down and tap "About phone"
  3. Find "Build number" and tap it 7 times
  4. You'll see "You are now a developer!" message
  5. Go back to Settings → "System" → "Advanced"
  6. Tap "Developer options"
  7. Enable "USB debugging"

Step 4: Using DroidCam with WeSee

  1. Start DroidCam app on your phone
  2. Connect phone to computer via USB
  3. Open DroidCam Client on computer and connect
  4. In WeSee, select "DroidCam USB" as camera source
  5. Choose your YOLO model and start detection!

Troubleshooting

Common Issues:
  • Connection failed: Check USB cable and debugging settings
  • No camera feed: Restart both DroidCam app and client
  • Poor quality: Use better lighting and stable phone position
Tips for Best Results:
  • Use good lighting for better object detection
  • Keep phone steady using a tripod or stand
  • Close other camera apps before using DroidCam
  • Use landscape orientation for wider view
